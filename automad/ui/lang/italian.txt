sys_cache_purge_info: In contrasto per semplicemente cancellare la cache, spurgo della directory della cache spostano anche tutti i file temporanei, le pagine eliminate, le pagine eliminate, gli aggiornamenti e le immagini memorizzate nella cache nella directory temporanea del sistema. Sulla maggior parte dei sistemi la directory temporanea verrà ripulita automaticamente di volta in volta o sul riavvio. Tuttavia, su alcune directory temporanee dei sistemi deve essere spuntata manualmente.
